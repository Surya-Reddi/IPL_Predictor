## Installation

1. Clone the repository:
```bash
[git clone https://github.com/YOU/IPL_Predictor.git](https://github.com/Surya-Reddi/IPL_Predictor.git)
cd IPL_Predictor
```

2. Install dependencies:
```bash
pip install -r requirements.txt
```

3. Run the app:
```bash
streamlit run app.py
```
